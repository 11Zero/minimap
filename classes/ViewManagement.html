    Object ::observeMinimaps(iterator) Source

    Calls iterator for each present and future minimap views. It returns a subscription Object with a off method so that it is possible to unsubscribe the iterator from being called for future views.

    Parameters:

    • (Function) iterator — A Function to call for each minimap view. It will receive an object with the following property:
      • view - The MinimapView instance
