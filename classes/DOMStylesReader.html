    Class DOMStylesReader

    Defined in: lib/mixins/dom-styles-reader.coffee
    Inherits: Mixin

    Overview

    This mixin is used by the {CanvasRenderer} in MinimapElement to read the styles informations from the DOM to use when rendering the Minimap.

    Instance Method Summary

    • String ::retrieveStyleFromDom(scopes, property, shadowRoot = true, cache = true) This function insert a dummy element in the DOM to compute its style, return the specified property, and remove the element from the DOM.
