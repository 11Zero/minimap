    Class CanvasDrawer

    Defined in: lib/mixins/canvas-drawer.coffee
    Inherits: Mixin

    Overview

    The CanvasDrawer mixin is responsible for the rendering of a Minimap in a canvas element.

    This mixin is injected in the MinimapElement prototype, so all these methods are available on any MinimapElement instance.

    Instance Method Summary

    • String ::getDecorationColor(decoration) Returns the background color for the passed-in decoration object.
    • String ::getDefaultColor() Returns the default text color for an editor content.
    • Number ::getTextOpacity() Returns the opacity value to use when rendering the Minimap text.
    • String ::getTokenColor(token) Returns the text color for the passed-in token object.
    • ::initializeCanvas() Initializes the canvas elements needed to perform the Minimap rendering.
    • ::updateCanvas() Performs an update of the rendered Minimap based on the changes registered in the instance.
