# Minimap package

A preview of the full source code, likes Sublime Text minimap.

![](https://github.com/fundon/atom-minimap/blob/master/resources/minimap-dark.png?raw=true)

### Features

* Redacted font, by @abe33
* Gray overlayer
* Supports mousewheel
